# Расписание турниров

## Winnergy

### Задача

Вам дано описание турнира в виде списка пар команд, которые будут играть друг против друга.

Каждая пара - это два числа, разделенные пробелом. Первое число - это номер команды, второе - номер команды, с которой она играет.

Команды пронумерованы от 1 до n.
Например, если n = 4, то турнир может выглядеть так:
```
1 2
2 3
3 4
4 1
```
То есть команда 1 играет с командой 2, команда 2 играет с командой 3, команда 3 играет с командой 4, команда 4 играет с командой 1.
Вам нужно определить, кто будет победителем турнира.
Победителем турнира считается команда, которая выиграет больше всех матчей.
Если несколько команд могут быть победителями, то выведите их все в порядке возрастания номеров.
### Решение
```python
def winnergy(n, matches):
    wins = [0] * n
    for match in matches:
        wins[match[0] - 1] += 1
    max_wins = max(wins)
    winners = [i + 1 for i in range(n) if wins[i] == max_wins]
    return winners
```
### Тестирование
```python
assert winnergy(4, [[1, 2], [2, 3], [3, 4], [4, 1]]) == [1, 2, 3, 4]
assert winnergy(4, [[1, 2], [2, 3], [3, 4], [4, 1], [1, 3], [2, 4]]) == [1, 2, 
3, 4]

, 3]]) == [1, 2, 3, 4]

```
### Сложность
O(n)
### Объяснение
Мы создаем список wins, в котором будем хранить количество побед каждой команды.

Затем мы проходим по списку матчей и увеличиваем количество побед команды, которая выиграла.

После этого мы находим максимальное количество побед и выводим все команды, которые выиграли это количество матчей.
